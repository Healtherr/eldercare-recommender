# 稳退规划：养老金融产品推荐系统

这是一个前端静态 MVP，可直接在浏览器中打开 `index.html` 使用。

## 已实现

- 面向 35-60 岁未退休人群的 MBTI 式养老规划问卷，共 13 题。
- 多维评分：退休期限、储备能力、风险承受、收入稳定、流动性需求、保障缺口、养老金缺口。
- 画像编码：长期/中程/临退，保守/均衡/增长，基础保障/保障补强。
- 产品库与匹配算法：根据问卷答案推荐储蓄、理财、基金、保险四类养老金融产品组合，并给出配置比例和启动清单。
- 响应式页面：桌面与手机均可使用。

## 文件

- `index.html`：页面结构
- `styles.css`：界面样式
- `app.js`：问卷、评分、推荐逻辑和渲染
